# ThreeJS

Nesse projeto eu fiz uma introdução a threeJS apenas carregando gltf e alterando alguns parametros apenas para fins de estudo.

Para utilizar o projeto é só utilizar 

### Instalar as dependências
`npm install`

### Rodar o projeto
`npm run dev`
